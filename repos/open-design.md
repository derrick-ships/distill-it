# open-design

**Source:** https://github.com/nexu-io/open-design  
**Product:** Local-first, agent-native design workspace for macOS/Windows. Alternative to cloud design tools — generates HTML prototypes, decks, images, and videos via AI agents with 150 brand-grade design systems.  
**Stack:** Node.js ~24, TypeScript, Hono, Electron, better-sqlite3, Next.js, pnpm monorepo  
**Version distilled:** v0.10.2  
**Date distilled:** 2026-06-16
