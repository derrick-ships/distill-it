# Domain: codegen

Systems that generate design artifacts — HTML prototypes, presentation decks, images, and videos — from natural language prompts via AI models. Covers multi-pathway generation, vendor API routing, artifact storage, and live preview rendering.

## Features in this domain

- [[design-artifact-generation--from-open-design]] — 4 pathways: HTML prototypes, HTML-based decks, images (via AIHubMix), and MP4 videos (Seedance/Wan/Veo/Sora)
