# Domain: git

Git operations that run inside the browser without a local git CLI. Features cover cloning remote repositories into browser-based sandboxes (using isomorphic-git), CORS proxies for bypassing cross-origin restrictions on git servers, and integrating version history into browser-native development tools.

## Repos studied
- bolt.diy — isomorphic-git clone into WebContainer via CORS proxy
