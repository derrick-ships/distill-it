# Domain: streaming

Real-time stream output, transcoding, re-muxing, and multi-destination delivery. Features in this domain cover taking live or file-based media and pushing it to network endpoints, converting formats on the fly, and orchestrating multi-stream broadcast pipelines.

## Repos studied in this domain

- **vlc** — `stream_out` module chain: transcode, duplicate, HTTP/RTP output, VLM scheduling

## From scira
- **scira** — [[resumable-streaming-search--from-scira]]: durable generation (Redis-buffered via ai-resumable-stream) decoupled from reconnectable delivery, with real mid-stream Stop.
