# Domain: runtime

In-browser and sandboxed code execution environments. Features in this domain cover running real server-side code (Node.js, Python, etc.) inside the browser tab using technologies like WebContainer, iframe sandboxes, and virtual file systems — without any cloud VM or backend.

## Repos studied
- bolt.diy — WebContainer-based full-stack runtime in the browser

## Related domains
- [[code-generation]] — code generation produces the files the runtime executes
- [[deployment]] — deployment pushes the runtime's output to production hosting
